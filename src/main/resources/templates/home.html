Spring Boot Project Info
Spring boot project admin username:admin, password:admin
Spring boot project manager username:manager, password:m
Spring boot project customer1 username:c1, password:z
Spring boot project customer2 username:c2, password:x
Project uses Spring Security, Spring MVC, Spring Data ,Hibernate, React, AWS Elastic Beanstalk, AWS S3, ReactJS
Security is provided by both Form-Based Authentication and JWT
JWT token is created at successful login
Connected to MySQL database on AWS
User passwords are encrypted before saving to the database
Admin is authorized to perform CRUD operations on Users and also authorized to access Manager section
When a customer is added by admin, customer is also automatically added to a seperate customer table for ordering purposes
Admin and Manager are authorized to perform CRUD operations on products, also add/edit product pictures
Add/Edit product picture functionality provided by ReactJS and Security is provided by JWT when editing product pictures
Product pictures are stored on AWS S3 Bucket and GET/POST requests secured by JWT
Admin and Manager are authorized to show Products database content
Customer is authorized to access Place Order and enter number of items to order and also see their own order history
Customers can see past orders either on a Thymeleaf or ReactJS rendered page
Customers can see place order either on a Thymeleaf or ReactJS rendered page
If order is successful, order date and time saved along with order data
Customer, customer info, order, orderitem and product saved into different tables and connected through foreign keys
